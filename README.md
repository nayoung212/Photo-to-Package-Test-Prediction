# Photo-to-Package-Test-Prediction

[ML] 반도체 Photo 공정 지표 기반 패키지 테스트 결과 예측

## 데이터
- Photo 공정 공정지표 F 데이터: 웨이퍼 생산 과정 중 노광(Photo) 단계에서 발생하는 각종 계측 및 설비 지표 데이터
- 구성: 각 단위별로 수집된 수십 개의 공정 변수와 최종 패키지 테스트 결과(정상/불량)를 포함함

## 목적
- 공정 효율화 및 수율 개선: 최종 패키징 공정까지 가기 전, 앞선 Photo 공정 단계의 데이터만으로 최종 품질(불량 여부)을 조기에 예측하고자 함
- 비용 절감: 불량이 예상되는 제품을 조기에 스크랩하거나 공정 조건을 수정하여 불필요한 후속 공정 비용을 최소화하는 모델 구축이 목표

## 분석
- 탐색적 자료 분석: 데이터 구조 확인 및 결측치 처리, 각 공정 지표별 통계량 및 상관관계 시각화를 통한 주요 변수 파악
- 데이터 전처리: 머신러닝 모델 학습이 가능하도록 데이터 스케일링, 인코딩, 이상치 제거 및 학습/검증 데이터 분리 수행
- 머신러닝 기반 분류: 전처리된 데이터를 바탕으로 최종 패키지 테스트 결과(PKT_Y)를 예측하는 이진 분류 모델 학습 준비
